Code Status
Code 0 : Diperlukan bantuan mendesak. Semua satuan tugas harus segera menghentikan apa yang sedang mereka lakukan, baik yang menetapkan tersangka atau terlibat dalam pengejaran aktif, dan menghadiri kode 0 sesegera mungkin. Tanda Kode 0 adalah saat petugas sudah down. Code 1 : Petugas dalam panggilan darurat. Code 2 : Merespon panggilan non-darurat artinya tanpa lampu dan sirine tapi jika ada apa-apa biarkan terus merespon, dan jika macet, nyalakan lampu dan sirene sejenak. Code 3 : Merespon panggilan darurat yang artinya dengan lampu dan sirine. Code 4 : Tidak diperlukan unit lagi. Code 5 : Seorang detektif/petugas menyuruh untuk menjauh dari area tersebut (Biasanya digunakan saat ada penyelidikan yang sedang berlangsung di suatu lokasi). Code 6 : Pemeriksaan di luar mobil (Investigasi). Code 7 : Istirahat, tidak bisa dihubungi.
